A work of interactive fiction.

Fetch Quest was created with Inform and has IFID 20F4A5A1-93CF-4244-937F-22B6BD0FE8FF. To play a work like this one, you need an interpreter program: many are available, among them Gargoyle for MacOS, Linux or Windows; Spatterlight for MacOS; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the Quixe interpreter. You'll need to have Javascript enabled on your web browser.
